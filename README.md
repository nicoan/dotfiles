# dotfiles
My configuration files

## Installing
The install script is made for debian-based systems (with apt) with Gnome installed

```
./install.sh
```

## Content
  - i3-gaps configuration
  - i3-blocks configuration
  - i3lock color
  - Arc-Dark GTK theme
  - Papirus Icon theme
  - Dmenu configuration
  - Rofi configuration
